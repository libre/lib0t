## Support 433Mhz
- Device RF433 Chacon
- Device RF433 Otio
- Device RF433 Vachet shutter


## Exemple 
Exemple for power ON device CHACON canal A1.
```
curl "http://ipforlib0t/CHACON/A1/ON"
```
